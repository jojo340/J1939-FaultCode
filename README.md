
# J1939 FaultCode Support

## English

J1939 FaultCode is an offline reference app for looking up J1939 SPN, FMI, DTC, and brand-specific fault code information.

If you need help, find an issue, or want to send feedback, please contact us by email.

**Support Email:** your-email@example.com

### What This App Provides

- SPN code lookup
- FMI definition lookup
- DTC lookup using SPN + FMI
- Brand-specific fault code reference
- Favorites and lookup history
- English and Chinese interface
- Offline access to built-in reference data

### Important Notice

The information in this app is provided for reference only. Vehicle diagnostics and repair decisions should always be verified with official manufacturer service manuals, diagnostic tools, and professional judgment.

# J1939 诊断码技术支持

## 中文

J1939 诊断码是一款用于查询 J1939 SPN、FMI、DTC 以及品牌故障码信息的离线参考工具。

如果你在使用过程中遇到问题，发现数据错误，或希望提出建议，可以通过邮件联系我们。

**技术支持邮箱：** your-email@example.com

### App 功能

- SPN 参数查询
- FMI 故障模式查询
- SPN + FMI 组合 DTC 查询
- 品牌故障码参考
- 收藏与查询历史
- 中文和英文界面切换
- 内置数据，核心查询功能可离线使用

### 重要说明

本 App 中的信息仅供维修和诊断参考。实际车辆故障诊断和维修决策，应结合原厂维修手册、专业诊断设备以及维修人员的专业判断。
